# Password Strength Checker (Python)

Simple CLI tool that evaluates a password and prints a score, rating, entropy estimate, and suggestions.

## Run

```powershell
python password_strength_checker.py
```

## Example Output

```text
Password Strength Checker
Enter a password (or press Enter to quit):
Score: 68/100
Rating: Good
Estimated entropy: 47.5 bits
Suggestions:
- Add symbols (e.g., !@#$%).
```
